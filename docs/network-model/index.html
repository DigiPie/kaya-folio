Network Model (CS4226)

Published on November 30, 2020

In this document, you will find my summary for the Network Performance Model and Queueing Model content covered under CS4226: Internet Architecture course taught by Dr. Richard Ma. I compiled this document with the help of notes written by my good friend Matthew over here.

Performance metrics​

MetricDefinition
Link Rate / BandwidthTheoretical maximum capacity; indirectly related to speed
Throughput / Effective bandwidthActual data transfer rate between source and destination; directly related to speed; with end-to-end delay factored in

End-to-end delay​

For a given packet being sent over a link, end-to-end delay is made up of:

  • Transmission delay: dependent on packet size L (bits)L\ (bits)L (bits) and link rate R (bits/s)R\ (bits/s)R (bits/s): dT=LRd_{T} = \frac{L}{R}dT​=RL​
  • Propagation delay: dependent on distance and propagation speed
  • Processing delay: dependent on LLL
  • Queueing delay: dependent on queue size and dTd_{T}dT​

Little's Law​

Wikipedia cite

The long-term average number LLL of customers in a stationary system is equal to the long-term average effective arrival rate λ\lambdaλ multiplied by the average time WWW that a customer spends in the system.

In the context of Internet Architecture:

L=λWL = \lambda WL=λW

  • LLL: Average number of packets in the system
    • L=lim⁡t→∞1t∫0tL(s)dsL = \lim_{t \to \infty} \frac{1}{t} \int^{t}_{0} L(s) dsL=limt→∞​t1​∫0t​L(s)ds
  • λ\lambdaλ: Average packet arrival rate for the system
    • λ=lim⁡t→∞N(t)t\lambda = \lim_{t \to \infty} \frac{N(t)}{t}λ=limt→∞​tN(t)​ where N(t)N(t)N(t) is the number of packets which arrived up to time ttt
  • WWW: Average sojourn time
    • W=lim⁡n→∞1n∑i=1nWiW = \lim_{n \to \infty} \frac{1}{n} \sum^{n}_{i = 1} W_{i}W=limn→∞​n1​∑i=1n​Wi​ where n is the number of packets and WiW_{i}Wi​ is the waiting/sojourn time for the iiith packet

Modelling arrival time​

Given:

  • tit_{i}ti​: Arrival time of iiith packet
  • TiT_{i}Ti​: Inter-arrival time where Ti≜ti+1−tiT_{i} \triangleq t_{i + 1} - t_{i}Ti​≜ti+1​−ti​

Model TiT_{i}Ti​s using independent and identically distributed (i.i.d.) random variable TTT which is exponentially distributed, with λ≥0\lambda \geq 0λ≥0 condition. Exponential distribution was chosen because of its memoryless property.

Properties:

  • P(T>t)=e−λtP(T > t) = e^{-\lambda t}P(T>t)=e−λt
  • P(T≤t)=1−e−λtP(T \leq t) = 1-e^{-\lambda t}P(T≤t)=1−e−λt
  • E[T]=1λE[T] = \frac{1}{\lambda}E[T]=λ1​
  • Memoryless property: P{T>s+t ∣ T>s}=P{T>t}P\{T > s + t\ |\ T > s \} = P\{T > t\}P{T>s+t ∣ T>s}=P{T>t}

Merging Poisson processes​

TiT_{i}Ti​s are i.i.d. random variables distributed as TTT with rate λ\lambdaλ. This arrival pattern is called a Poisson process, in which starting time does not matter (memoryless property). Therefore, two Poisson proccesses can be merged to create a new Poisson process:

  • λ=λ1+λ2\lambda = \lambda_{1} + \lambda_{2}λ=λ1​+λ2​
  • T=min{T1,T2}T = min\{T_{1}, T_{2}\}T=min{T1​,T2​}
  • P(T>t)=P(T1>t∧T2>t)=e−λ1te−λ2t=e−(λ1+λ2)tP(T > t) = P(T_{1} > t \wedge T_{2} > t) = e^{-\lambda_{1} t} e^{-\lambda_{2} t} = e^{-(\lambda_{1} + \lambda_{2}) t}P(T>t)=P(T1​>t∧T2​>t)=e−λ1​te−λ2​t=e−(λ1​+λ2​)t
  • E[T]=1λ1+λ2E[T] = \frac{1}{\lambda_1 + \lambda_2}E[T]=λ1​+λ2​1​

M/M/1 queueing model​

  • Single server with a queue of infinite length
  • Exponential i.i.d. service time with rate μ\muμ

Definitions​

  • Utilization: ρ=λμ\rho = \frac{\lambda}{\mu}ρ=μλ​
    • Percentage of time whereby the server is busy (1 item in server)
      • Alternatively, mean number of packets in the server
    • Derived from E[L]=λE[Ws]E[L] = \lambda E[W_{s}]E[L]=λE[Ws​] where average service time = E[Ws]=1μE[W_{s}] = \frac{1}{\mu}E[Ws​]=μ1​
    • For server to be stable, ρ=λμ<1⇒λ<μ\rho = \frac{\lambda}{\mu} < 1 \Rightarrow \lambda < \muρ=μλ​<1⇒λ<μ
  • πi\pi_{i}πi​: Percentage of time or probability that there are exactly iii packets in the system
    • πi=P{L=i}=ρi(1−ρ)\pi_{i} = P\{L = i\} = \rho^{i}(1 - \rho)πi​=P{L=i}=ρi(1−ρ)
    • Follows a Geometric distribution

Burke's Theorem​

Wikipedia cite

Burke's theorem asserts that for a M/M/1 queue in the steady state with arrivals, a Poisson process with rate parameter λ\lambdaλ:

  1. The departure process is a Poisson process with rate parameter λ\lambdaλ.
  2. At time ttt the number of packets in the queue is independent of the departure process prior to time ttt.

  • ρi=λμi\rho_{i} = \frac{\lambda}{\mu_{i}}ρi​=μi​λ​
