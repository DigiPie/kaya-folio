MongoDB Cheatsheet

Created on December 23, 2020

This document was migrated from DigiDocs

The purpose of this page is to summarise key mongodb commands you will often use.

Print a collection's documents#

# Select the databaseshow dbsuse <db-name>
# Show all collections in the databaseshow collections
# Print out all documents in the databasedb.<collection-name>.find()
# Print out in an easy-to-read but less compact formatdb.<collection-name>.find().pretty()

Delete documents#

show dbsuse <db-name>
# Show all collections in the databaseshow collections
# Remove all documents in the collectiondb.<collection-name>.remove( { } )
# See reference for more information
